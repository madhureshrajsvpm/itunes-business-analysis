# Apple iTunes Business Analysis

## Project Overview
This project provides a comprehensive analysis of the Apple iTunes store dataset (2017-2020). The goal was to identify sales trends, customer behavior patterns, and employee performance to provide actionable business recommendations.

## Tools Used
* **SQL (MySQL):** Data cleaning and complex querying for business metrics.
* **Tableau:** Interactive dashboard creation for stakeholder visualization.
* **Microsoft Word:** Formal business reporting and executive summary.

## Key Insights
* **Revenue:** Total generated revenue of $4,709.43, with 2019 being the peak year.
* **Market Dominance:** Rock music accounts for over 55% of total sales.
* **Customer Loyalty:** 100% of customers are repeat buyers, averaging 10 purchases each.
* **Top Region:** Prague, Czech Republic emerged as the highest-performing regional hub.

## Project Components
1. **[SQL Analysis](./sql/iTunes_Analysis.sql):** Includes scripts for data cleaning, schema modification, and trend analysis (Monthly revenue, genre popularity).
2. **[Interactive Dashboard](./dashboard/):** A Tableau workbook visualizing global sales distribution and artist performance.
3. **[Executive Report](./reports/Business_Report.pdf):** A detailed business document outlining the "Why" behind the data and strategic recommendations for growth.
